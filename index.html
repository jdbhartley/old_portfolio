About Me

I'm James Hartley from Southern California, I'm currently developing independent games for PC, iOS and Android. Always actively looking for artists and other developers to collaborate with. This website is a showcase for some of the projects I have been working on.

My Skillset

Unity3D

I have been using Unity3D since version 3.5 and love the engine. The games you see here were developed in Unity3D. I'm planning my next project in Unreal however just to try it out. I always love learning new software.

Programming

I've been coding since I was 14 and started with Visual Basic 6. I'm almost entirely self-taught. My favorite language is C#, however I'm familiar with Java, JavaScript, PHP, HTML, CSS and Currently studying C++ to broaden my understanding of programming and get a grasp on memory management.

Art and Music

Almost all of the music in my games is created by myself, I have had a passion for music since a young age and play a few instruments as well as create music digitally. I have dabbled before in 3D Modelling and have a fondness for pixel art. My recent titles art has been my own work (excluding Tafl)

Tafl - A Kings Pass (Windows)

Description

Originally planned as an iOS and Android game, this one was never oficially released. However I have a build here for you to download and play. The game is based on an ancient viking boardgame called Hnefetafl. You can play a local multiplayer match however the servers are no longer running for the multiplayer mode. The game was scrapped after ~almost completion. during playtesting with friends I decided it was just simply boring to play. So I moved on to another project. Lesson learned. always playtest while in prototype stage.
